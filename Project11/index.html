Introduction

A programming language is a system of notation for writing computer programs.Programming languages are described in terms of their syntax (form) and semantics (meaning), usually defined by a formal language.

Languages usually provide features such as a type system, variables, and mechanisms for error handling. An implementation of a programming language is required in order to execute programs, namely an interpreter or a compiler. An interpreter directly executes the source code, while a compiler produces an executable program.

  • Computer architecture has strongly influenced the design of programming languages, with the most common type developed to perform well on the popular von Neumann architecture.
  • While early programming languages were closely tied to the hardware, over time they have developed more abstraction to hide implementation details for greater simplicity.

C and Java

C and Java are both popular programming languages, but the key difference is that C is a procedural language focused on low-level system programming and direct hardware control, while Java is an object-oriented language prioritizing platform independence and ease of use, making it better suited for general application development; essentially, C gives you more control over hardware while Java abstracts away many complexities for easier coding.

C is a compiled language that is it converts the code into machine language so that it could be understood by the machine or system. Java is an Interpreted language that is in Java, the code is first transformed into bytecode and that bytecode is then executed by the JVM (Java Virtual Machine).

Variables

You use variables as symbolic names for values in your application.The names of variables, called identifiers, conform to certain rules.

In JavaScript, a variable is a named container that stores values. Variables can store numbers, strings, and objects. You can use variables to store information that you might need to use or modify again.

A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($); subsequent characters can also be digits (0-9). Because JavaScript is case sensitive, letters include the characters "A" through "Z" (uppercase) and the characters "a" through "z" (lowercase).
